Operadores Aritiméticos

  • + -- soma
  • - -- subtração
  • / -- divisão
  • * -- multiplicação
  • ** -- potência e.g. '3**3 = 9'
  • pow(x,y) -- potência e.g. 'pow(3,3) = 9' diferente do operador acima (**), este é ignorado na ordem de cálculo aritmética (1º ^, 2º *, 3/4º +/-) onde parênteses se resolvem primeiro

    • OBSERVAÇÃO: A raiz de um número pode ser extraída elevando o número ao valor que corresponde á metade do seu ídice(raiz).
    Exemplo: 9**0.5 = 3
              a


  • % -- módulo, retorna o resto da divisão e.g. 10%3 = 1
  • // -- omite a parce fracionária (decimal) de uma divisão não inteira (resto) e.g. 10//3 = 3, quando o resultado deveria ser uma dízima periódica: 3.3333333333333335
  • ( ) -- expressões entre parênteses serão, naturalmente, executadas primeiro e.g. 5*(2+20) => 22*5 = 110, quando o resultado correto seria 30 pois 5*2+20 => 10+20


Texto(strings)

  • "\n" --pula uma linha (equivalente ao '< br >' no HTML
  • "\t" --tab (4 espaços)
  • print(f'-- necessário para a concatenação de variáveis em strings, adicionando o nome da variável desejada envolto por chaves ({nome_variável:02})(: requirido para indicar formatação de texto adicional e o '02' indica o número de casas, :.2 indica o número de casas decimais quando o número é um float)
  • .find(< target >)-- retorna o padrão argumentado, caso não seja encontrado, retorna 'False'
  • count(< string >)-- retorna a quanitade de caracteres em uma string

    • REPLACE

Formatação de texto com strings

  • {: ^ X < string >} --centraliza o texto em X casas decimais
  • {: < X < string >} --centraliza o texto à esquerda em X casas decimais
  • {: > X < string >} --centraliza o texto à direita em X casas decimais
  • print('...string', end='') --determina o que será inserido no final da linha que uma função print( ) imprime. Caso seja deixado vazio, a quebra de linha automática imediata ao print() subsequente não será implementada
  • print('string_qualquer', sep='(conteudo)'-- determina o caractere a ser adicionado entre uma palavra e outra, se não especificado, o padrão é um espaço em branco (_)
  • string.upper() --converte o texto da string em CAIXA ALTA
  • string.lower() --converte o texto da string em caixa baixa
  • string.title() --converte a primeira letra de cada palavra em CAIXA ALTA
  • string.capitalize() --converte a primeira letra da string em CAIXA ALTA
  • string.strip() --remove quaisquer espaços extras à direita e esquerda da string e.g. em uma entrada pelo usuário
  • string.lstrip() --remove quaisquer espaços extras à esquerda da string e.g. em uma entrada pelo usuário
  • string.rstrip() --remove quaisquer espaços extras à direita da string e.g. em uma entrada pelo usuário
  • string.split() --cria uma lista a partir de uma string onde cada item é separado por um espaço na string
  • string.join() --une a string em uma só e.g. frase = 'O L Á' => '_espaço_'.join(frase) => 'O_espaço_L_espaço_Á'

  • print(f'{listagem[pos]:.<30}') --- nessa linha, o ' : ' informa ao interpretador que a string será formatada, o ' . ' imprime o caractere . até o final do número de casas especificado com ó número '30', e o ' < ' informa ao interpretador que o texto será alinhado à esquerda. < ^ > estes caracteres alinham o texto à equerda, centralizado e direita respecitivamente


Listas

  • nome_lista.insert(index, item) -- insere um item em index (int)
  • nome_lista.append(conteúdo/variável) -- insere item no final
  • nome_lista.pop(index) -- remove item e armazena na memoria (pode ser colocado em uma variavel designando o comando para o valor de uma)
  • del nome_lista[index]-- remove um item da lista/dicionário [nºX]
  • nome_lista.remove(item)-- remove o item pelo nome (procurando e removendo em seguida)
  • sorted(nome_lista)-- organiza a lista/dicionário em ordem alfabética temporariamente (utilizando o parâmetro reverse=True, a ordem é invertida)
  • len(nome_lista)-- retorna o tamanho de uma lista/string em caracteres
  • nome_lista.reverse( )-- organiza a lista em ordem alfabética reversa permanentemente
  • nome_lista.sort( )-- organiza a lista em ordem alfabética permanentemente (utilizando o parâmetro reverse=True, a ordem é invertida)
  • nome_array/string.count(algarismo)-- retorna o número de vezes que um (algarismo) objeto está contido num array/string
  • nome_array/string.index(algarismo, posição à iniciar a contagem de)-- retorna a posição em que o (algarismo) está contido num array/string. Se o segundo parâmetro for omitido, inciará a partir do index 0
  • enumerate(objeto, start=X) -- retorna um objeto enumerado contatenando o numerador em uma tupla no formato (enum, item). O objeto deve ser um int, um array ou qualquer outro objeto que suporte iteração.
    Exemplo:
    >>> estacoes = ['Primavera', 'Verão', 'Outono', 'Inverno']
    >>> list(enumerate(estacoes))
    [(0, 'Primavera'), (1, 'Verão'), (2, 'Outono'), (3, 'Inverno')]
    >>> list(enumerate(estacoes, start=1))
    [(1, 'Primavera'), (2, 'Verão'), (3, 'Outono'), (4, 'Inverno')]

  • sum(nome_lista)-- retorna a soma de todos os números da lista.
  • max(nome_lista)-- retorna o maior número da lista.
  • min(nome_lista)-- retorna o menor número da lista.

    • OBSERVAÇÃO: Os itens que nao modifiquem diretamente o valor contido em um array do tipo tupla (e.g. sum, min, max) também podem ser usados em tuplas


  • Também é possível adicionar tuplas ou dicionário dentro de uma lista:
    >>> dicionario = {'nome':'Erick', 'num': 123}
    >>> lista = [dicionario, 'string 1', 'int 2', 'float 3']
    >>> lista
    >>> [{'nome':'Erick', 'num': 123}, 'string 1', 'int 2', 'float 3']


Dicionários

  • items(nome_dicionário)-- retorna uma lista de pares < chave >:< valor > de um determindo dicionário
  • keys(nome_dicionário)-- devolve uma lista de todas as chaves do dicionário.

    • "Percorrer as chaves, na verdade, é o comportamento-padrão quando percorremos um dicionário com um laço, portanto este código produzirá a mesma saída se escrevêssemos:
    for name in favorite_languages:
    em vez de:
    for name in favorite_languages.keys():"

  • values(nome_dicionário)-- devolve uma lista de todos os valores do dicionário.


Operadores Lógicos

  • and - -- analisa duas condições simultaneamente e só retorna 'True' se as duas forem 'True'
  • or - -- analisa duas condições simultaneamente e retorna 'True' se pelo meno uma das condições forem verdadeiras
  • not - -- pode se comparar à not False: pass| / usado para inverter uma booleana, muda de 'True' para 'False' e de 'False' para 'True'
  • is - -- checa se dois objetos tem o mesmo ID (id(x)), mesmo valor na memória
  • is not- -- o inverso de < is >
  • in - -- checa se o objeto está contido em (array, string...)
  • not in - -- checa se o objeto NÃO está contido em (array, string...)


Operadores de comparação

  • == -- igual a, analisa o conteúdo de duas variáveis e retorna 'True' se o conteúdo for igual, e 'False' se o conteúdo for diferente (não confundir com o operador único =, este é usado na criação de variáveis para se atribuir um valor especícfico.)
  • != -- diferente de, analisa o conteúdo de duas variáveis e retorna 'True' se o conteúdo for diferente, e 'False' se o conteúdo for igual
    • -->
  • > -- maior que, analisa o conteúdo de duas variáveis e retorna 'True' se o conteúdo for maior, e 'False' se o conteúdo for menor
  • < -- menor que, analisa o conteúdo de duas variáveis e retorna 'True' se o conteúdo for menor, e 'False' se o conteúdo for maior
    • =-->
  • >= -- maior ou igual, analisa o conteúdo de duas variáveis e retorna 'True' se o conteúdo for maior ou igual, e 'False' se o conteúdo for menor
  • <= -- menor ou igual, analisa o conteúdo de duas variáveis e retorna 'True' se o conteúdo for menor ou igual, e 'False' se o conteúdo for maior
  • is -- analisa se dois objetos são idênticos na memória comparando o ID de cada objeto e retorna 'True' caso sim, e 'False' caso não. O ID de cada objeto pode ser obtido através do comando: id(< objeto ou variável >)
    O mesmo que id(a) == id(b)


Declarações que Python avalia como 'False'

  • 'False'
  • 'None'
  • Número Zero -- de qualquer tipo numérico | Exemplo: 0, 0.00
  • Qualquer sequência vazia -- Exemplo: '', [], (), {}

    • Todo o resto será avaliado como 'True'
